#  仿网易云音乐移动端项目

## 技术栈

Vue全家桶+Vant组件库

## 项目运行

### 前端部分

#### 安装依赖

```
npm install
```

#### 运行项目
```
npm run serve
```

### 后端部分

Github地址：https://github.com/Binaryify/NeteaseCloudMusicApi
